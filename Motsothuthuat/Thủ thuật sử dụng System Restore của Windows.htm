Thủ thuật sử dụng System Restore của Windows
 

Không cần phải cài đặt thêm những phần mềm hỗ trợ phục hồi, sửa lỗi hệ thống nếu tận dụng hết khả năng vốn có của Windows. Windows XP và ME cung cấp cho bạn khả năng rất hữu ích đó là khôi phục hệ thống trở lại chính xác trạng thái tại một thời điểm trong quá khứ.

Nó đặc biệt quan trọng trong trường hợp bạn cài đặt thêm những chương trình phức tạp dẫn đến làm lỗi một số file hệ thống. Tuy nhiên nó bỏ qua sự khôi phục đối với các file thông dụng như đuôi .doc hoặc .jpg vì bạn chỉ muốn khắc phục lỗi hệ thống chứ không phải muốn lãng phí toàn bộ công sức của bạn đối với các file văn bản cũng như file ảnh về trạng thái chưa được xử lý.

Mở System Properties bằng cách vào Start -->Settings --> Control Panel --> System. Cách thứ 2 là kích phải chuột vào biểu tượng My Computer ngay trên màn hình rồi chọn Properties. Tiếp theo chọn tab System Restore

 

Bỏ chọn ở checkbox "Turn off System Restore on all drives", khi đó bạn sẽ thấy phần Status của các ổ cứng trong mục Available Drives sẽ chuyển từ trạng thái "Turned off" sang "Monitoring". Nói chung ổ C là ổ mặc định của Windows chứa thông tin về hệ điều hành và các chương trình cài đặt, những trục trặc xảy ra thường chỉ liên quan đến ổ C (trừ khi bạn tuỳ chọn cài đặt sang các ổ khác trong quá trình cài đặt).

Có thể tắt chế độ Monitoring riêng cho từng ổ để tiết kiệm bộ nhớ bằng cách, chọn ổ muốn bỏ trạng thái Monitoring rồi kích vào Setting, khi đó sẽ hiện lên cửa sổ "Drive X: Settings" như sau:

 

Tăng, giảm dung lượng bộ nhớ dành cho việc khôi phục bộ nhớ bằng cách rê thanh trượt sang bên trái để giảm, và bên phải để tăng.

Kích chọn ở hộp kiểm "Turn off System Restore on this drive", rồi chọn OK, bạn sẽ thấy hộp thoại sau hiện lên

 

Chọn Yes để hoàn tất quá trình.

Các bước để thực hiện khôi phục hệ thống

- Vào Start --> Programs --> Accessories --> System Tools --> System Restore

 

- Chọn "Restore my computer to an earlier time", rồi kích vào "Next", hộp thoại tiếp theo hiện ra yêu cầu bạn lựa chọn thời điểm bạn muốn khôi phục (Restore Point), ví dụ hôm nay là ngày 30/10, bạn chọn ngày 29/10 là thời điểm máy bạn chưa gặp trục trặc.

 

- Chọn Next tiếp để đến hộp thoại khẳng định lại yêu cầu lần nữa. Sau đó máy sẽ tự động tiến hành quá trình khôi phục. Sau khi khởi động lại máy là quá trình đã hoàn tất.

Chú ý:

Chức năng System Restore có trạng thái mặc định là "Not configured" Bạn có thể kích hoạt để chuyển sang trạng thái "Enable" bằng cách truy cập tới Group Policy: Vào Start --> Run , đánh: "gpedit.msc". Local Computer Policy --> Computer Configuration --> Administrative Templates --> System --> System Restore.

 

Chọn Turn off System Restore, kích phải chuột, chọn Properties --> Enable.

 

Tuy nhiên nếu các bước bạn thực hiện được bình thường thì không cần thiết phải quan tâm sửa đổi các thông số theo hướng dẫn của chú ý trên.
